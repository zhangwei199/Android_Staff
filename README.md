# 员工管理系统APP
>本项目是一个能与web服务器端连接操作MySQL数据库来管理员工信息的安卓软件。

>软件客户端共有6个活动页面，分别是用户登录、员工信息列表、添加、修改删除、用户注册、和管理员操作的页面

>安卓客户端通过http来访问web服务器端的Servlet来实现用户登录注册操作、对员工信息进行增删改查以及管理员操作的功能。

> 想了解更多详情请观看本项目其他文档
